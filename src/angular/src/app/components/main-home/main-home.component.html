Sistema de Gestão de Reembolsos

Início

Chegou o novo Sistema de Gestão de Reembolsos!

Esta nova atualização está carregada de novidades e trás consigo uma interface gráfica moderna, simplificada e altamente focada no fluxo de trabalho do operador.

Este projeto de código aberto foi promovido pela Direção Regional de Saúde, é mantido pela datapoint.software e tem o repositório oficial publicado em Github.
